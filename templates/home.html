Zero Trust Hospital Management System

This prototype secures healthcare infrastructure using Zero Trust Architecture, encryption, intrusion detection, and activity monitoring.

Key Principle: No user, device, or request is trusted by default. Every sensitive request is verified each time before access is granted.

System Modules

  • Role-based login for Admin, Doctor, and Nurse
  • Zero Trust verification on patient-data requests
  • IoT device verification
  • Patient data encryption
  • Intrusion detection and security logging
